## 1. 软件概述

### 1.1 软件名称及版本
**软件名称**：Smart QC Tools 工业级质量控制软件  
**版本号**：V1.0  
**功能模块**：散点分析与相关性研究模块 (Scatter Analysis Module)

### 1.2 开发背景与用途
散点图（Scatter Plot）是研究两个或多个数值变量间相关性的关键手法。本模块针对工程场景下变量关系复杂（如温度与收缩率、压力与强度）的特点，开发了 **趋势回归平滑技术**。相比传统图表工具，本软件支持三维气泡模式（加入权重 Z 轴），并能实时预测因果关系强弱，为解决复杂的质量变异问题提供数据支撑。

## 3. 核心功能及操作规程

### 3.1 二维与三维数据录入 (Multi-Dimension Entry)
**功能描述**：支持单一因果（X-Y）或带有权重的影响因素（X-Y-Z）分析。

**操作规程**：
1. **坐标轴定义**：在控制面板输入 `XAxis` (如：加热温度) 和 `YAxis` (如：抗拉强度)。
2. **气泡模式开启** (可选)：若输入数据包含第三个维度（Z），需定义 `ZAxis`。系统将通过气泡直径展示第三变量的影响程度。
3. **数据格式**：
   - 二维：`- 10.5, 20.1` (表示 X=10.5, Y=20.1)
   - 三维：`- 10.5, 20.1, 5` (Z=5 决定气泡大小)

### 3.2 线性回归趋势线 (Linear Regression Focus)
**功能描述**：通过最小二乘法算法实现趋势拟合。

**操作规程**：
- 在 DSL 中设置 `ShowTrend: true`。
- 系统将计算并绘制一条黄金分割趋势线，穿过数据最为密集的区域。
- 用户可通过 `Color[Trend]` 调整趋势线颜色，以增强视觉对比度。

### 3.3 画布交互控制 (Visual Console)
- **基准大小调节**：使用 `Size[Base]` 控制原始散点的直径（建议 4-8px）。
- **透明度调节**：配置 `Opacity`（0.1-1.0）。在数据点极其密集（点重叠）时，调低透明度可清晰观察“数据聚集区”的浓度分布。

## 4. 技术特性亮点

1. **高维数据映射核心**：自研的 Scale 线性映射算法，可精确地将物理数值转换为屏幕像素点坐标，确保坐标轴比例严谨。
2. **毫秒级实时渲染引擎**：支持在一秒内处理并绘制数千个散点坐标，具备良好的大数据量吞吐性能。
3. **专业级坐标轴算法**：自动识别数据的动态范围（Min/Max），并生成美观的刻度标签。
